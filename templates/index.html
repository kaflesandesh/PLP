Welcome to the Personalized Learning System

This system provides a personalized learning experience for students, instructors, and administrators.

Features Include

User Management: Handles students, instructors, and admin users.
Course Management: Manages course creation, enrollment, and content.
Learning Material: Handles access to learning materials and assignment submissions.
Feedback and Progress: Manages student progress tracking and feedback.
System Utilities: Includes reporting, bug tracking, and system maintenance.
